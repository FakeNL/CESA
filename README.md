### CESA

#### Running
Clone the project:<br>
`git clone git@github.com:FakeNL/CESA.git`

Install bundler:<br>
`gem install bundler`

Install rails:<br>
`gem install rails`

Run migrations:<br>
`bin/rake db:migration`

Install dependencies:<br>
`bin/bundle install`

Run the project:<br>
`bin/rails s`
